# Інструкція для розробників

## Архітектура проекту

Проект використовує простий підхід з трьома основними файлами:

### index.html
Основна HTML-структура додатку з елементами:
- Поле вводу для нових завдань
- Кнопки фільтрації
- Список завдань
- Лічильник активних завдань

### style.css
Стилі для всіх елементів інтерфейсу:
- Responsive дизайн
- Стилі для різних станів завдань
- Кольорова схема

### script.js
Вся логіка додатку:
- Управління масивом завдань
- Рендеринг інтерфейсу
- Обробка користувацьких дій
